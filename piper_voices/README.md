# Piper Voice Assets

This folder is reserved for optional local Piper TTS voice assets.

Jarvis uses Edge TTS by default. Piper assets are only needed if you configure `voice_mode=piper` or set `TTS_PROVIDER=piper` with a valid `PIPER_MODEL_PATH`.

Large local voice model files may be kept here for local use. Do not commit private or licensed voice assets unless they are safe to redistribute.
